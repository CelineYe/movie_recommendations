Movie Recommendation System
=====================================================
Setting up the environment :

Requirements: Python 2.7, numpy, pandas, scikit-learn

For web system:
  --Install Django
          pip install django

  -- Bootstrap support for Django projects
        pip install django-bootstrap3
        https://pypi.python.org/pypi/django-bootstrap3

  -- Install django-registration
        pip install django-registration
        https://django-registration.readthedocs.io/en/2.2/install.html

=====================================================

Run the recommendation engine on local:
       1. Extract files in a lcation on your computer.
       2. Find the fold which contains manage.py
       3. Run command under this fold :  python manage.py  run server  localhost:8000
       4. Go to http://localhost:8000/review
